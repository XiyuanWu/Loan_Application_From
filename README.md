# Loan Application From

## Project Overview

LendSwift — Multi-Step Loan Application is a front-end portfolio project that simulates a real-world digital lending experience for a fictional Indian fintech brand. The app guides users through an 8-step wizard covering loan selection, personal details, KYC, address, employment, optional co-applicant, document upload, and final review.

Built with React 19, Vite, Tailwind CSS, React Hook Form, and Zod, the project focuses on complex form UX: conditional steps, cross-field validation, simulated PAN/Aadhaar verification, PIN-based address lookup, encrypted auto-save, and accessibility best practices. There is no backend — all data stays in the browser, and submission returns a locally generated reference number.

This repository is intended as a public showcase of front-end engineering work: clean component architecture, schema-driven validation, responsive UI, and broad test coverage with Vitest and Cypress.
